# Getting Started

> 클론 후, 각 backend/frontend 폴더 내에서 yarn install과 yarn start를 실행해 주세요


### clone Repository
git clone https://github.com/kimhaein/Todo-App.git

### backend

1. cd backend
2. yarn install
3. yarn start

start -> localhost:5000

### frontend

1. cd frontend
2. yarn install
3. yarn start

start -> localhost:3000
